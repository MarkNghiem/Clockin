## *GET /welcome*
- Request Parameters: `n/a`
- Request Option: `n/a`
- Request Header:
```
{
  "Content-Type": "application/json",
}
```
- Request Body: `n/a`
- Response:
```
{
  "message": "Welcome to Clockin'",
}
```
- Response Codes:
```

400 Bad Request
404 Not Found
500 Internal Server Error
```
